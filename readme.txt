== Description ==

A simple and easy way to broadcast your WordPress post to LCIMedia. LCIMedia is a cloud based tool that enables you to quickly deliver your wordpress content into a personalized tablet app.

== Installation ==

1. Upload to your plugins folder, usually found at 'wp-content/plugins/'
2. Activate the plugin on the plugins screen
3. Set LCIMedia feed url from Settings menu.
